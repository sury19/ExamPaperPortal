# Email Configuration - Resend (Primary)

Instructions:
1. Replace "re_your_resend_api_key_here" with your actual Resend API key (starts with "re_")
2. Add these lines at lines 16-17 in your .env file
3. You can keep your existing Gmail SMTP configuration as fallback
4. Restart your backend server after adding

Note: Resend will be used as primary email service, Gmail SMTP will be used as fallback.
